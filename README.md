# Employee Management Application

## Purpose:
A basic mobile application that an employer could use to create employee schedules.

## Libraries:
- React
- React-Native
- Firebase
- Redux
- Redux-Thunk
- Lodash
- React-Native-Communications
- React-Native-Router-Flux
